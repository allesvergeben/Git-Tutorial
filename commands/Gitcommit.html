Kommando: git commit

Wofür nutze ich commits?

Ein Commit entspricht einem Entwicklungsstand einer Arbeit. Der Vorteil von Commits ist, dass man zu jederzeit auf einen alten Commit, bzw. Entwicklungsstand zugreifen kann.

Wie verwende ich git commit?

Man verwendet den Befehl git commit, um einen neuen Commit anzulegen.

Schritt 1

Unser Repository weist voerst nur zwei Commits auf, möchte man einen dritten hinzufügen, braucht man den Befehl git commit.

Schritt 2

Unser Repository weist nun 3 Commits auf.
