Kommando: rebase

Wofür nutze ich rebase?

Bei dem Befehl Rebase handelt es sich um eine Alternative zu merge; mit Hilfe von Rebase kann man die Commits eines aktuellen Branch auf das Ende eines anderen Branch verschieben. Dadurch wird der Commit Verlauf im Gegensatz zu dem Befehl merge linearisiert.

Wie verwende ich git rebase?

Man verwendet den Befehlt git rebase. um die Commits eines aktuellen Branch auf das Ende eines anderen Branch zu verschieben.

Schritt 1

Unser Repository weist voerst nur zwei Commits auf, diese sollen zusammengefügt werden. Dafür muss man zuerst den Befehl git checkout master in die Konsole eingeben.

Schritt 2

Als zweiten Schritt gibt man den neu erlernten Befehl git rebase bugFix ein.
